# Playwright Learning Notes

## Useful Commands

```bash
npx playwright test          # Run all tests
npx playwright test --ui     # Run with UI mode
npx playwright codegen       # Record tests
npx playwright show-report   # View last report
```

## Mac Shortcuts for DevTools

| Action | Shortcut |
|---|---|
| Open DevTools | `Cmd+Opt+I` |
| Console | `Cmd+Opt+J` |
| Inspect element | `Cmd+Shift+C` |
| Reload (hard) | `Cmd+Shift+R` |
| Next breakpoint | `F8` |
| Step over | `F10` |
| Step into | `F11` |

## VS Code Playwright Extension

- `Cmd+Shift+P` → "Playwright: Run test"
- Click the test playwright icon in the sidebar
